# Industrial Machine Diagnostic System

Industrial Machine Diagnostic System adalah aplikasi berbasis web
yang digunakan untuk menganalisis kondisi mesin industri berdasarkan
parameter suhu, getaran, suara, dan beban mesin.

## Fitur Aplikasi
- Input data kondisi mesin
- Analisis kondisi mesin menggunakan OOP
- Penentuan status kondisi mesin
- Penyimpanan dan riwayat analisis

## Teknologi yang Digunakan
- Node.js
- JavaScript
- Object-Oriented Programming (OOP)
- Sequelize ORM
- EJS Template Engine

## Cara Menjalankan Aplikasi
1. Install Node.js
2. Clone repository
3. Jalankan perintah `npm install`
4. Jalankan aplikasi dengan `npm start`
